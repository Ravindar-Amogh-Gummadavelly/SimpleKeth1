# 1) 🎯 VISUAL STYLE

### Design Direction

* Modern + clean + nature-inspired
* Warm, grounded, and trustworthy
* Functional-first (clarity over decoration)

---

### Mood & Feel

* Calm and reassuring
* Farmer-friendly (not tech-heavy)
* Confident decision-making

---

### Core Principle

👉 *“Show the decision clearly, not the complexity behind it.”*

---

# 3) 🔤 TYPOGRAPHY

### Font Family

* Headings → *Sora*
* Body → *DM Sans*

---

### Font Scale

* H1 → 52px (Hero)
* H2 → 34px (Section titles)
* H3 → 24px (Card titles)
* Body → 17px
* Small → 14px

---

### Typography Rules

* Use *bold + large text* for decisions
* Keep text minimal and readable
* Use spacing for clarity (line-height: 1.5–1.7)

---

### Special Styling

#### Decision Text

* ALL CAPS
* Bold
* Slight letter spacing

Example:
👉 *SELL NOW*

---

#### Profit Display

* Large + bold
* Highlight color (green)

Example:
👉 *₹12,450*

---

# 4) 📐 LAYOUT STYLE

---

## Structure

* Fullscreen vertical sections
* Smooth scroll experience

---

## Grid System

* 12-column layout
* Max width: 1200px
* Centered content

---

## Spacing

* Section padding: 80–120px

---

## Responsiveness

* Mobile-first design
* Stack layout vertically on small screens
* Large touch-friendly elements

# 5) 🧩 COMPONENTS

---

## 🔘 Buttons

* Rounded (10–12px radius)
* Green background + white text
* Hover: darker + slight scale

---

## 🧾 Cards

* White / cream background
* Soft shadow
* Rounded corners (12px)
* Padding: 20–24px

---

## 📊 Decision Card (CORE COMPONENT)

Displays:

* Recommendation → *SELL NOW / HOLD*
* Expected Profit → ₹ value (large)
* Best Mandi → location

👉 This is the *most important UI element*

---

## 📍 Input Fields

* Dropdown (crop selection)
* Number input (quantity)
* Location input

Style:

* Clean borders
* Large clickable areas

---

## 🔔 Alerts

* Small banners/cards
* Color-coded:

  * Green → opportunity
  * Red → urgency
  * Yellow → caution

---

## 📈 Graphs

* Simple line charts
* Minimal, clean, readable

# 🔥 FINAL DESIGN PRINCIPLES

---

### 1. Decision First

👉 Always highlight:

* SELL NOW / HOLD
* Profit
* Best location

---

### 2. Simplicity Over Features

👉 Don’t overload UI with data

---

### 3. Visual Clarity

👉 Big text + clear colors + simple layout

---

### 4. Farmer-Friendly UX

👉 Easy to read
👉 Easy to click
👉 Easy to understand

---

### 5. Controlled Animation

👉 Use animation only to enhance—not distract
